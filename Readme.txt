# How to implement

## Backend development workflow

```json
virtualenv env
source env/bin/activate
pip install -r requirements.txt
python manage.py runserver
```
## Frontend development workflow

```json
npm i
npm start
```

## For deploying

```json
pip install -r requirements.txt
npm run build
python manage.py collectstatic
python manage.py runserver 
```
